### 扩展语法（不是所有的MarkDown编辑器都支持）
#### 上标
```
30^th^
```

#### 下标
```
H~2~O
```
H~2~O

#### 缩略
```
*[缩略]: 看到了吗 我出现了
*[空格]:  字符之间要有空格
鼠标移到 缩略 就会有文字显示
字符之间要有 空格 才生效
没有空格就没有生效
```
